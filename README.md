# Sensor Monitoring 

Frontend dashboard built with React to visualize sensor data from the backend API.

## Tech Stack

- React  
- Vite  
- Nginx (Production)  
- Docker  

## Features

- Displays sensor readings in table format  
- Pagination support  
- Alerts view  
- Integrated with FastAPI backend  
- Dockerized production build  

## Run with Docker

docker compose up --build  

Frontend:  
http://localhost:3000  

## Development

npm install  
npm run dev  

## Production Build

npm run build
